# Tkinter To-Do List

A simple desktop To-Do List application built using Python and Tkinter.

This project allows users to manage daily tasks through a clean graphical interface.

---

## Features

- Add new tasks
- Mark tasks as completed
- Update existing tasks
- Remove tasks
- Scrollable task list
- Keyboard shortcuts
  - **Enter** → Add task
  - **F11** → Maximize window
  - **F12** → Restore window

---

## Built With

- Python 3
- Tkinter
